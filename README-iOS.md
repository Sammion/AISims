# 房地产大亨 iOS App 安装指南

## 📱 方法一：PWA 快速安装（推荐，无需开发者账号）
无需编译，直接在iPhone上安装使用：

1. 打开iPhone自带的Safari浏览器
2. 访问游戏地址：`http://你的服务器IP:38856/`
3. 点击底部的「分享」按钮
4. 选择「添加到主屏幕」
5. 输入App名称「房地产大亨」，点击「添加」
6. 回到桌面即可看到App图标，点击即可像原生App一样使用

**优点：**
- ✅ 无需开发者账号
- ✅ 无需越狱
- ✅ 5秒完成安装
- ✅ 支持离线运行
- ✅ 功能完整，和原生App体验一致

## 🏗️ 方法二：原生IPA包安装（需要苹果开发者账号）

### 前置要求
- 苹果开发者账号（99美元/年）
- Xcode 14+
- 配置好开发者证书和描述文件

### 编译步骤
1. 进入项目目录：
   ```bash
   cd /root/.openclaw/workspace/real-estate-life-game
   ```

2. 打开Xcode项目：
   ```bash
   npx cap open ios
   ```

3. 在Xcode中配置：
   - 选择你的开发团队
   - 修改Bundle ID为你自己的
   - 选择你的测试设备
   - 点击运行即可安装到手机

4. 导出IPA包：
   - 修改 `ios-build.sh` 中的 `YOUR_TEAM_ID` 为你的团队ID
   - 运行构建脚本：
     ```bash
     chmod +x ios-build.sh
     ./ios-build.sh
     ```
   - 生成的IPA文件在 `build/ipa/` 目录下

### 安装方式
1. **TestFlight 分发（推荐）：**
   - 将IPA上传到App Store Connect
   - 添加测试人员，通过TestFlight安装

2. **企业级分发：**
   - 企业开发者账号可以直接分发IPA
   - 用户通过描述文件安装

3. **越狱设备：**
   - 直接安装IPA文件即可

## 🐛 常见问题
1. **PWA无法安装？**
   - 确保使用Safari浏览器
   - 确认网络可以正常访问游戏地址
   - 尝试刷新页面后再添加

2. **原生App编译失败？**
   - 确认已安装最新版Xcode
   - 检查开发者证书配置是否正确
   - 运行 `npx cap sync ios` 重新同步项目

3. **安装后无法打开？**
   - 企业级分发需要在「设置→通用→VPN与设备管理」中信任证书
   - TestFlight安装需要接受测试邀请
