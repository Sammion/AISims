# 🏘️ 房地产大亨 - 人生模拟游戏

一款基于真实房地产行业发展历史的人生模拟游戏，玩家从90年代的初中毕业开始，通过一次次人生选择，最终成为叱咤风云的房地产大亨，或普通的地产从业者。

## ✨ 特色功能

### 🎭 高度自定义
- **角色创建**：自定义姓名、性别、颜值
- **家庭背景**：多种家庭条件选择（普通/小康/富裕/富商）
- **父母性格**：不同的父母性格影响人生发展

### 📅 真实时代背景
- **时间线**：从15岁初中毕业到65岁退休，完整人生周期
- **时代还原**：真实还原90年代至今房地产行业发展历程
- **历史事件**：包含楼市暴涨、政策调控、行业转型等真实事件

### 🔀 多分支剧情
- **6个职业生涯阶段**：中学→大学→职场初期→上升期→巅峰期→后期
- **30+剧情事件**：覆盖学习、工作、创业、结婚、贵人相助等关键节点
- **随机事件系统**：每次游戏体验都不同

### 📊 成长系统
- **五大属性**：知识、社交、健康、声望、财富
- **属性影响**：不同属性影响选择的成功率和事件结果
- **财富系统**：从万元级到亿元级的真实资产增长体验

### 🏆 丰富结局
- **7种不同结局**：从普通从业者到房地产大亨
- **真实人物参考**：结局参考王健林、潘石屹、左晖等地产名人经历
- **人生回顾**：完整的人生历程记录，可回溯所有关键选择
- **Wiki链接**：结局提供参考人物的维基百科链接

### 💾 存档功能
- **多存档管理**：支持多个游戏进度保存
- **本地存储**：数据持久化，刷新页面不丢失

## 🎮 快速开始

### 在线体验
1. 打开浏览器访问：`http://你的服务器地址:38856/`
2. 点击「开始游戏」创建你的角色
3. 在每个事件中做出选择，体验不同的人生

### iOS安装（PWA方式，推荐）
1. 用iPhone的Safari浏览器访问游戏地址
2. 点击底部「分享」按钮 → 选择「添加到主屏幕」
3. 回到桌面即可像原生App一样使用

### 本地运行
```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 生产构建
npm run build

# 预览生产版本
npm run preview
```

### 原生iOS App编译
```bash
# 安装Capacitor依赖
npm install @capacitor/core @capacitor/cli @capacitor/ios

# 添加iOS平台
npx cap add ios

# 同步项目
npx cap sync ios

# 打开Xcode项目
npx cap open ios
```

## 🛠️ 技术栈
- **前端框架**：Vue 3 + Composition API
- **构建工具**：Vite
- **UI组件库**：Element Plus
- **状态管理**：Pinia
- **路由**：Vue Router
- **移动端适配**：Capacitor
- **存储**：LocalStorage

## 🎯 游戏玩法说明
1. **角色创建**：设置你的初始属性和家庭背景
2. **事件选择**：每个年龄阶段会出现不同的事件，做出你的选择
3. **属性成长**：每个选择会影响你的属性和财富
4. **人生发展**：从初中毕业开始，经历高考、工作、创业、结婚等人生大事
5. **结局判定**：65岁退休时根据你的总财富、声望、能力等判定最终成就
6. **人生回顾**：查看你这一生的所有重要选择和经历
